1. PROJECT OVERVIEW & GOALS
------------------------------------------------------------------------
RakshaSutra is designed as a command centre application targeting cyber
fraud, counterfeiting, digital arrest scams, and network-based financial
crimes. Built using React, Vite, Leaflet, D3.js, and Chart.js, it uses a 
highly interactive, glassmorphism-themed dark-mode UI to present deep
intelligence insights suitable for a national-level hackathon.

NOTE ON DATA: This platform is a frontend-only interactive demonstration.
All AI analysis, transaction networks, geospatial incidents, and verification
reports are simulated using pre-configured mock data (JSON fixtures) to showcase
real-world UI flows and mock integration capabilities without live backend APIs.

------------------------------------------------------------------------
2. SYSTEM ARCHITECTURE & COMPONENTS DEFINED
------------------------------------------------------------------------
- Five-Layer Event-Driven Architecture (Showcased on Architecture page):
  1. Intake & Edge Layer (Vite, WhatsApp API, IoT, Call Ingest)
  2. Event Streaming Layer (Apache Kafka)
  3. Processing & Analytics Layer (Redis, PostGIS, TimescaleDB, Neo4j)
  4. Core Intelligence Engines (indicBERT, RawNet3, YOLOv8, GraphSAGE)
  5. Presentation Layer (React SPA Command Centre Dashboard)

- Core Platform Modules:
  1. Scam Sentinel (Real-time voice and transcript spoof detection)
  2. NoteSure (Curreny verification scan matching UV & watermark features)
  3. FraudGraph (Visualizing fraud rings & transaction networks using D3.js)
  4. CrimeMap (Geospatial heatmap and hotspot tracking with Leaflet)
  5. NagrikShield (Citizen-facing chatbot assistant for reporting and lookup)
