1. PROJECT OVERVIEW & GOALS
------------------------------------------------------------------------
RakshaSutra is designed as a command centre application targeting cyber
fraud, counterfeiting, digital arrest scams, and network-based financial
crimes. Built using React, Vite, Leaflet, D3.js, and Chart.js, it uses a 
highly interactive, glassmorphism-themed dark-mode UI to present deep
intelligence insights suitable for a national-level hackathon.

------------------------------------------------------------------------
2. SYSTEM ARCHITECTURE & COMPONENTS DEFINED
------------------------------------------------------------------------
- Five-Layer Event-Driven Architecture (Showcased on Architecture page):
  1. Intake & Edge Layer (Vite, WhatsApp API, IoT, Call Ingest)
  2. Event Streaming Layer (Apache Kafka)
  3. Processing & Analytics Layer (Redis, PostGIS, TimescaleDB, Neo4j)
  4. Core Intelligence Engines (indicBERT, RawNet3, YOLOv8, GraphSAGE)
  5. Presentation Layer (React SPA Command Centre Dashboard)

- Core Platform Modules:
  1. Scam Sentinel (Real-time voice and transcript spoof detection)
  2. NoteSure (Curreny verification scan matching UV & watermark features)
  3. FraudGraph (Visualizing fraud rings & transaction networks using D3.js)
  4. CrimeMap (Geospatial heatmap and hotspot tracking with Leaflet)
  5. NagrikShield (Citizen-facing chatbot assistant for reporting and lookup)

3. STEP-BY-STEP DEVELOPMENT PROGRESS & CHANGELOG
------------------------------------------------------------------------

[STEP 1] Initialization & Theme Setup
- Initialized React SPA with Vite framework.
- Built global design system in `src/index.css` defining HSL color variables, 
  glassmorphism panels, spacing system, custom scrollbars, and fonts.
- Implemented background mesh gradient animations and micro-interactions 
  (hover lift, breathing status dots, scan lines, fade-in animations).
- Created brand logo and visual layout with Sidebar & TopBar navigation.

[STEP 2] Mock Data Fixtures Creation
- Created detailed synthetic databases inside `src/data/`:
  - `scamSessions.js`: Live call transcripts, caller IDs, audio features, and key scam phrases.
  - `fraudNetwork.js`: Graph database simulation (nodes for bank accounts, SIM cards, targets).
  - `crimeEvents.js`: 200 geospatial crime logs covering Indian cities with loss amounts and severity levels.
  - `currencyData.js`: Scan reports matching 6 critical note features (watermark, intaglio, UV, etc.).
  - `chatScenarios.js`: Multilingual conversations (12 languages) simulating citizen chatbot flows.
  - `analyticsData.js`: Data for charts, metrics, weekly trends, and system layers.

[STEP 3] Shared Utilities & Global Layout
- Built circular SVG ConfidenceGauge for visual spoof confidence tracking.
- Created StatusBadge indicator component for safety levels.
- Formulated the TopBar live clock, activity pulse, and breadcrumb layout.
- Structured the Sidebar containing segregated navigation levels (Overview, Modules, System).

[STEP 4] Core Page Implementation
- Command Centre Dashboard: Integrated interactive Chart.js widgets (Line chart for trends, Doughnut chart for threat types, Bar chart for state loss). Included an animated Data Pipeline flow tracking ingest-to-alert cycles and an Intelligence Fusion tab showing cross-module correlation logic.
- Scam Sentinel Page: Renders list of live suspect call sessions. Includes an interactive transcript viewer highlighting malicious vocabulary, audio spoof confidence sub-gauges, and rapid enforcement action buttons.
- NoteSure Verification Page: Demonstrates mobile image scanner viewport. Added an animated progress bar tracing watermark, security thread, micro-lettering, and UV luminescence. Keeps an immutable verification ledger audit trail.
- FraudGraph Intelligence Page: Renders D3.js Force-Directed Graph tracking interconnected fraud networks. Nodes are draggable and filterable (Person, Phone, Bank Account, SIM Card). Clicking nodes populates side intelligence logs with option to export intelligence packages.
- CrimeMap Geospatial Page: Deployed Leaflet Map showing nationwide crime density. Integrated Leaflet.heat overlay, interactive circle markers mapping specific FIR events, filtering systems (UPI Fraud, Digital Arrests, OTP Phishing), and top state ranking lists.
- NagrikShield Citizen Page: Designed messaging interface with scenario quick-start prompts (e.g. "Got a CBI call", "Suspect ₹500 note"), automated AI typing responses, risk verdicts, emergency helpline buttons, and 12-language localization switch.
